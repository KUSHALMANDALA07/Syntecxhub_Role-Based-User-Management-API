# Syntecxhub_Role-Based-User-Management-API

## 📌 Project Overview

This project is a **Role-Based User Management API** built using **Node.js, Express.js, and MongoDB** as part of my Backend Development Internship at Syntecxhub.

The API implements **authentication and role-based authorization**, where users can have different roles such as **admin** and **user**, and access to endpoints is controlled based on roles.


## 🚀 Features

* User Registration & Login
* JWT Authentication
* Role-Based Access Control (RBAC)
* Middleware for authentication & authorization
* Admin-only protected routes
* Promote user to admin
* Block users
* Secure API structure


## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT (jsonwebtoken)
* bcryptjs
* dotenv
* cors
